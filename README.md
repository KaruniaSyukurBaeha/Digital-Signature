## About The Project

ViTrash is an advanced application designed to classify garbage into six categories: Battery, Plastic, Paper, Metal, Organic, and Glass. It leverages state-of-the-art deep learning techniques and modern web technologies to promote sustainable waste management and environmental conservation.

## Features
* AI-Powered Garbage Classification: Built with TensorFlow, Keras, and Vision Transformers for high accuracy.
* Real-Time Webcam Integration: Utilize the webcam for real-time garbage detection.
* Cross-Platform Support: Responsive design for seamless use on desktop and mobile.
* Secure Backend: Built using Flask with secure connections via NGINX and Cloudflare.
